# 🎬 Movie Search — Android приложение для поиска фильмов

Это полноценное Android-приложение для поиска фильмов с поддержкой пагинации, уведомлений, темной темы, пробного периода и динамического онбординга через Firebase.

---

## 📌 Для чего создан проект?

Проект создан как **обучающее портфолио**, демонстрирующее профессиональную разработку на Android. Он показывает, как сочетать современные библиотеки и архитектурные подходы в реальном приложении:

- 🔹 Работа с REST API (TMDb)
- 🔹 Бесконечная прокрутка (`Paging 3`)
- 🔹 Локальное хранение данных (`Room`, `DataStore`)
- 🔹 Фоновые задачи (`WorkManager` + напоминания)
- 🔹 Темизация: день/ночь, автоматическое переключение по батарее
- 🔹 Разделение на flavours: бесплатная и платная версии
- 🔹 Пробный период и динамические промо-экраны через Firebase Remote Config
- 🔹 DI с Dagger, MVVM, Coroutines + RxJava
